# Urban Traffic Control Digital Twin

A smart-city simulation for evaluating adaptive traffic-signal behavior under variable demand and incidents.

## Highlights
- Multi-intersection adaptive phase control.
- Incident injection and congestion propagation effects.
- Real-time KPI tracking (queue, speed, delay, emissions proxy).
- Visual digital twin rendering on Canvas.

## Stack
- HTML5 Canvas
- Vanilla JavaScript simulation
- Responsive CSS dashboard

## Run
Open `index.html` in your browser.
